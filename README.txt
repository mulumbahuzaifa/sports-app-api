# PeerLearn-APIs

Acessing the APIs via https://immense-badlands-08984.herokuapp.com/

    	How To Run The Project On Your Local Machine
- Clone the project to your system
- run cd PeerLearn-APIs/venv/Scripts
- run activate.bat
- Navigate back to PeerLearn-APIs root folder
- run pip install -r requirements.txt
- run flask run

	Available URLS
- /login (requires username & pass as parameters)
- /allCourses 
- /createUser (requires username, firstname, lastname, email, pass as parameters)
- /getEnrolledCourses (requires username & pass as parameters)
